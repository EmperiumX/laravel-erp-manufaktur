# ERP Manufaktur & Konsinyasi - CV. New Citra Indonesia

Aplikasi *Enterprise Resource Planning* (ERP) berbasis web yang dirancang khusus untuk mengelola siklus bisnis pabrik makanan, mulai dari pembelian bahan mentah, produksi (Bill of Materials), hingga penjualan konsinyasi dan kasir.

## 🚀 Fitur Utama
- **Role-Based Access Control (RBAC):** Superadmin, Admin, Produser, dan Sales (Menggunakan Spatie).
- **Master Data:** Kelola Supplier, Toko/Mitra, Bahan Baku, dan Produk.
- **Dynamic Pricing:** Harga jual otomatis menyesuaikan kategori toko (Mitra, Agen, End User, dll).
- **Inventory Engine:** Pencatatan stok otomatis (Auto-deduct) dengan riwayat mutasi (Audit Trail).
- **Manufacturing / Produksi:** Sistem Bill of Materials (BOM) & kalkulasi HPP otomatis.
- **Purchasing & Sales:** Purchase Order (PO), Pengiriman Konsinyasi (DO), Retur, dan Direct Sales (Kasir).
- **Reporting:** Export laporan penjualan dan stok ke Excel, dan cetak Surat Jalan/Nota PDF & Direct Print (Kertas Kasir).
- **Dashboard Analytics:** Ringkasan pendapatan, pengeluaran, dan alert stok menipis secara *real-time*.

## 🛠️ Tech Stack
- **Framework:** Laravel 11
- **Frontend:** Laravel Breeze (Blade Template) + Tailwind CSS + jQuery (AJAX Dynamic Forms)
- **Database:** MySQL
- **Packages:** Spatie Permission, Laravel Excel, Barryvdh DomPDF
